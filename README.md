# Music Tags

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

## Production

```bash
npm run build
```

## Using as widget

```html
<div id="#baas-widget"></div>
<script src="{link}/com.elsten.bliss.baas.widget.js"></script>
<script>
  baasWidget.init({
    target: "#baas-widget"
  }); 
</script>
```
